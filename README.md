# Dogfight
Using a high level WebGL library Three.js and bullet physics engine javascript port Ammo.js.

# Controls:
- W - Pitch down
- S - Pitch up
- A - Roll left
- D - Roll right
- Q - Yaw left
- E - Yaw right
LeftShift - Throttle up
LeftCTRL - Throttle down
